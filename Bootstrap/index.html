Conclusions

Bootstrap offers open source components whcih allow us to design very cool and beautiful webpages.It makes our life easier because these kind of elements would take too long and much more code in HTML+CSS+JS by themselves. Finally, there are many solved questions on forums about many different troubles you can face while you are using Bootstrap; I will keep developing with it.
